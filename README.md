# Level Design Teaching Pack

用意されたプレイヤー・地形・敵・ギミックを使って、3〜5分程度の3Dステージを制作するためのUnity教材です。

## 対応環境

- Unity: `6000.3.19f1`
- Render Pipeline: Universal Render Pipeline `17.3.0`
- Input System: `1.19.0`
- AI Navigation: `2.0.13`

## 学生の開始手順

1. `LevelDesignTeachingPack_v1.0.0.zip`をダウンロードする。
2. ZIPを展開する。
3. Unity Hubの`Add`または`Open`から、展開したプロジェクトフォルダを選択する。
4. 初回インポートが終わるまで待つ。
5. `Assets/00_Scenes/02_StudentTemplate.unity`を開く。
6. StudentTemplateを複製して制作を開始する。

## 基本操作

| 操作 | 入力 |
|---|---|
| 移動 | W / A / S / D |
| カメラ | マウス |
| ジャンプ | Space |
| 現在の復帰地点へ戻る | R |

## 注意

- `Library`、`Temp`、`Logs`は配布物に含まれていません。初回起動時に再生成されます。
- Consoleに赤いErrorが出た場合は、制作を続ける前に教員へ報告してください。
